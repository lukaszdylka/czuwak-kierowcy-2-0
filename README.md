# 🚗 Czuwak Kierowcy 2.0

Progresywna aplikacja webowa (PWA) dla kierowców jeżdżących nocą.

## 🎯 Funkcje
- ⏰ Przypomnienia o czujności
- 🚨 System alarmowy
- 🛣️ Integracja GPS
- 📱 Działa offline

## 🚀 Użyj aplikacji
[Otwórz Czuwak Kierowcy 2.0](https://TWOJA-NAZWA.github.io/czuwak-kierowcy-2-0)
